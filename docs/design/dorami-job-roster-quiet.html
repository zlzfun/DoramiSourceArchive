设计推导:点名册(muster sheet)

参数四波收敛后,任务编辑的本体 = 勾人 + 定额 + 一个时刻。原「目录 → 编排单」双面板让 同一批节点显示两遍、右栏为一个数字消耗整行——形态随功能过时。点名册把勾选与配置合到同一行: 勾到谁,谁的行尾出现「上限」输入;未勾的行保持安静。模态由 1020px 收窄到 680px。

关键交互

  • 行尾定额:值 = 节点默认时 muted 灰字,与默认不同时 ink 加粗(安静的 diff 信号); 聚焦即变 ink 可编辑;点击输入框不切换勾选(stopPropagation);
  • 组头 sticky(surface 底)+ hover 浮现「全选」(该组整组编入;已全选时变「清空」);
  • 工具行:搜索(过滤列表,组头随空组隐藏)+「已编入 N / 总数 · 改动 M」mono 计数;
  • 头部时刻区沿用班次单已验证设计,窄幅重排:kicker+启用 / 名称大字 / cron+说明 / 人话回显+距下次;
  • 取消勾选不清空该节点已改的上限值(会话内保留,重勾即恢复——防误触丢配置)。

与现实现的关系

  • 替代 docs/design/dorami-job-editor-quiet.html 的双面板体(其头部设计保留);数据层 (jobDraft/toggleDraftFetcher/updateDraftParam/保存校验)不变,纯渲染形态更换;
  • 若未来某 preset 重新长出第二个参数:行勾中后可下挂一条参数行(点名册行支持展开), 形态可增长、不必回退双面板。
