# 阅读器重构(按靛版设计样页) —— ☑ 实施完成,待目检(2026-07-18)

> 性质:**用户面阅读器前端重构方案 + 执行记录**。触发:用户拍板设计样页
> (从零设计的「晨报 × 档案室」四带式布局 × 静默仪器配色版)后指示「基于样页进行用户面阅读器的前端重构」。
> 样页文件:scratchpad `reader-redesign-sample-indigo.html`(Artifact 已发布)。
> 实施决策:设计敏感、集中在 ReaderTab.jsx + index.css 单主战场,亲力亲为不拆包(沿用体验二波决策)。

## 样页 → 实现的映射原则

- **布局/按钮/动效按样页**,配色即现有 `--dorami-*` token(样页靛版本就是按 token 实值调的,实现直接引用 token,暗色自动成立)。
- **机制保留**:全屏壳数学(`html:has(.reader-shell)` 视口处理)、scroll-timeline 进度线、折叠把手、
  `reader-enter`/`reader-list-enter` 动画、hover 预取、骨架延迟、竞态防护——全部沿用,只换皮不换骨。
- **逻辑层(hooks/API 调用)不动**:未读体系、收藏、翻译/摘要缓存、日报拉取、订阅管理原样保留。
- 新增衬线字役 token `--font-serif`(Georgia/宋体):只出现在阅读窗大标题、日报报头、正文小节标题。

## 结构变更(四带式)

- ☑ **视图轨(新第一带,52px)**:今日 / 文章 / 动态 / 收藏 / 搜索。
  内容形态轴(article|bulletin)从中栏 mini-seg 升格为一级导航;「今日」= 混合流(不传 shape,新增 `'all'` 档);
  收藏从中栏星标钮升格为视图;搜索钮切换中栏搜索框显隐。选中任一源时视图轨按钮点击即清源回聚合。
- ☑ **源栏(232px)**:头部「我的订阅 · N 未读」;去掉「我的订阅」大导航行(聚合入口=视图轨「今日」);
  订阅行维持单行(名称+未读徽标+悬停减号),仍按 文章来源/动态来源 分组(不引入样页的 官方/媒体 编辑分组——
  现有 category 值域不受控,形态分组与视图轨轴一致且零风险);发现更多来源保留。
- ☑ **条目列(372px)**:列表头 = 标题+「N 篇·M 未读」小计 + mini-seg[全部|未读] + 全部标读图标钮;
  **日期分组**(今天/昨天/MM-DD,sticky 小节头,C 档转正);列表底色降为 soft(选中卡浮白+sh-1,弃 indigo 竖条);
  卡片结构改样页式:首行(源 LogoMark+源名+[动态 chip]+时间+未读点),标题,一行摘要;
  **卡上悬浮收藏星退场**(收藏动作收敛到阅读窗动作组);日报置顶卡改「报头」形态(wash 底+双细线+衬线刊名)。
- ☑ **阅读窗**:新增顶部 **crumb 工具条**(源 LogoMark+源名 | 动作图标组:原文/收藏/标读/**「译」字钮**),
  动作组从文章头部上移至此(常驻可见);文章头部 = mono kicker(源·类型)+ **衬线大标题** + mono 元信息行;
  AI 总结卡改样页 wash 形态;日报正文渲染**报头样式**(衬线居中刊名+日期);
  新增**上一篇/下一篇**页脚导航(真实沿当前列表序);进度线机制不变。
- ☑ **问哆啦美 fab**:改样页墨色胶囊、常显标签(弃 hover 展开)。
